⚡ AlgoPath — Algorithm Study Optimizer
A web-based project demonstrating Greedy and Dynamic Programming algorithms applied to a real-world study planning problem.

🧠 Algorithms Used
1. Greedy — Dijkstra's Shortest Study Path
Problem: Find the optimal path between two study topics with minimum difficulty cost
Algorithm: Dijkstra's shortest path — always greedily picks the unvisited node with lowest cost
File: greedy.js
2. Dynamic Programming — 0/1 Knapsack Study Planner
Problem: Given limited study hours, select the best topics to maximize total importance score
Algorithm: 0/1 Knapsack — bottom-up DP table with backtracking
File: dp.js

💡 Problem Statement
Students need to study smartly:

What is the easiest path from a beginner topic (Arrays) to an advanced topic (Dynamic Programming)?
Which topics should I study given only 10 hours, to maximize my preparation score?
